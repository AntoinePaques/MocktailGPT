# @mocktailgpt/ts

> TypeScript SDK and mock toolkit for OpenAI/ChatGPT APIs.
> Fully typed, contract-first, and DX-friendly.

---

## 🚀 Build
pnpm build

👉 Compile `src/` → `dist/` (ESM, types).

---

## 📦 Publish
npm publish --access public

👉 Ne publie que `/dist/`, `package.json`, `README.md` (grâce au .npmignore).

---

## 📜 Variables d’environnement

Chargées via `.env.local` à la racine du monorepo.
